# Repository Scope

This repository is a child component of the PQID publication stack. Its scope is
limited to the Scientific Data manuscript package:

- manuscript draft files;
- manuscript figures and source diagrams;
- figure-generation and descriptive-analysis scripts;
- compact public summary tables;
- submission checklist and repository-level documentation.

The repository intentionally excludes:

- raw PQID dataset splits;
- internal/no-license rows;
- private funding or publication-planning documents;
- unrelated manuscript drafts;
- Word lock files and temporary exports;
- local absolute-path traces.

For dataset access, use the Hugging Face dataset and Zenodo archive linked from
the root `README.md`.
